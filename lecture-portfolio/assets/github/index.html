Digital Design Lecture

Curricular themes, learning modules, and academic resources.

MODULE 01

Vector Mechanics & Balance

Exploring the pure geometry of visual anchors. Focuses on Bézier curve mathematics, concentric anchors, and symmetric balance. Analysed via *Plate I (Target Dynamics)*.

  • Concentric visual weights
  • Bézier handle optimization
  • Geometric focus structures
Lecture Slide PPTX Download ↓
MODULE 02

Isometric Spatial Theory

Translating 2D interfaces into 3D environments. This module examines vector-based isometric projection, isometric grids, and gradient ambient occlusion. Analysed via *Plate II (Digital Portal)*.

  • 30-degree projection planes
  • Axonometric depth illusions
  • Synthesized linear gradients
Syllabus PDF Download ↓
MODULE 03

Typographic Architecture

Merging icons with letterforms. Discusses path clipping, circular text alignments, glyph modification, and the semiotics of identity systems. Analysed via *Plate III & IV (Siren's Chronicle & Apple)*.

  • Text-on-path geometry
  • Ligature and kerning theory
  • Symmetry and shape intersection
